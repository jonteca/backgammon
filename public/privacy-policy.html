Privacy Policy

Last updated: 2 March 2026

1. Information We Collect

When you use the Backgammon Discord Activity, we receive your Discord user ID and username solely to manage game sessions. This information is held in memory during gameplay and is not stored permanently.

2. How We Use Information

Your Discord user ID is used only to:

  • Assign you to a game (black or white pieces)
  • Identify your moves during a multiplayer session

3. Data Storage

We do not store any personal data. Game sessions exist only in server memory and are deleted when all players disconnect or after a period of inactivity.

4. Third Parties

We do not share any data with third parties. The App communicates only with Discord's API for authentication purposes.

5. Children's Privacy

The App does not knowingly collect information from children under 13. Use of the App is subject to Discord's own age requirements.

6. Changes

We may update this policy at any time. Changes will be reflected on this page with an updated date.

7. Contact

Questions? Reach out via the project's GitHub repository.
